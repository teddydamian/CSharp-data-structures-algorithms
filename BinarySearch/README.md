# Binary Search
### Author: Teddy Damian

## Problem Domain
Given an array of integers and a new number. Output index location of the number if it exist in array. If not, return -1.

| Input | Expected Output |
| :----------- | :----------- |
| [2,3,4,5] 3 |  | 1
| [2,99,120,200] 12  | -1 |
